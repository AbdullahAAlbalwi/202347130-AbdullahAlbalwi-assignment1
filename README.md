# 202347130-AbdullahAlbalwi-assignment1

## Project Description
This project is a personal portfolio website built for Assignment 1.  
It includes:
- A short "About Me" section
- A "Projects" section
- A contact form layout
- A light/dark theme toggle saved in `localStorage`

The site is implemented with plain HTML, CSS, and JavaScript.

## Setup Instructions (Run Locally)
This is a static web project, so no package installation is required.

1. Open a terminal in the project root folder.
2. Start a local server:
   - Python: `python -m http.server 8000`
3. Open your browser and go to:
   - `http://localhost:8000`

You can also open `index.html` directly, but using a local server is recommended.
